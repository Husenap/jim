# Project Architecture

This is a Next.js+React application with:

- Routes in `/src/app`
- Components in `/src/components`
- Convex database schema in `/src/convex`
- Utils in `/src/utils`

## Coding Standards

- Use TypeScript for all new files
- Follow the existing naming conventions

# External Dependencies

Our app is based on these libraries:

- [Next.js documentation](https://nextjs.org/docs)
- [Convex documentation](https://docs.convex.dev/home)
- [Convex Ents documentation](https://labs.convex.dev/convex-ents)
- [Hero UI library documentation](https://v3.heroui.com/)
- [Tailwind css documentation](https://tailwindcss.com/)

When implementing new features, reference these patterns.
